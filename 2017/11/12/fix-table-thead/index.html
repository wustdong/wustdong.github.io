Css Transform 固定表头

发表于 2017-11-12 | 分类于 Angular2

项目中需用表格展示查询结果，表格数据内容较多时，不利于查看明细，因此需要在滚动内容时固定表头。

实现逻辑：给 table 注册滚动检测事件，有滚动时，将表头 translate（移动）回到原来的位置。

  • scrollTop: 这个元素的顶部到它的最顶部可见内容（的顶部）的距离的度量。当一个元素的内容没有产生垂直方向的滚动条，那么它的 scrollTop 值为0。
  • CSS3 transform 属性: transform 属性向元素应用 2D 或 3D 转换。该属性允许我们对元素进行旋转、缩放、移动或倾斜。
  • translate: 通过 translate(x,y) 方法，元素从其当前位置移动，根据给定的 left（x 坐标） 和 top（y 坐标） 位置参数。
