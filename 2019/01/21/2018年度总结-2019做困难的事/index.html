2018年度总结---2019做困难的事

发表于 2019-01-21

2018 年总结

2019年1月过了一大半了，才终于开始写18年的总结，姑娘今年也真的「年方」二八了。

一、2018 年看的书：

看完的：

  • 《小顾聊绘画：文艺复兴》
  • 《这里是没有我们存在的地方》
  • 《围城》
  • 《房思琪的初恋乐园》

没看完的：

  • 《最好的告别》
  • 《我的天才女友》
  • 《小岛经济学》
  • 《何为良好生活》
  • 《娱乐至死》
  • 《亲密关系》
  • 《断舍离》
  • …

看书多是电子书，除了划线和想法，没怎么做笔记。然后看过就忘记了。
18年过的真快，整理书籍的时候，才发现好多我以为18年看的书其实是17年末看完的，18年没怎么看书。最有意思的应该是发现 iphone 自带的图书 app支持的.epub格式的书，这种原生支持的书籍看起来就感觉很舒服，因为不需要依附其他软件。

二、2018 年值得一提的事：
  • 存了点钱（仅相对于学生时代，却仍是穷苦的底层人民中的一员；
  • 断断续续的跑步和瑜伽；
  • 开始了解经济学，购买了薛兆丰的《薛兆丰的经济学课》；
  • 听完了陈立老师的《陈立客厅》；
  • 喜欢上了听《不丧》和《晓说》。《不丧》是一对在美国教书的年轻小夫妻联合抵制当下流行的丧文化推出的随性谈话博客，一方面他们沟通交流的方式，让我觉得特有意思；另一方面每个月在他们组建的微信群会有一个读书分享，就好像是个很远的朋友；姨妈期间我最喜欢听他们的播客，轻松自在，又飘在人间。《晓说》是高晓松主持的一档文化类节目，讲各种各样的故事，听起来比较过瘾。加上他又是《奇葩说》的导师，所以就觉得比较亲切。洗漱的时候听得比较多。
  • 做了个手术，所幸现在还活蹦乱跳的；
三、2018 年的遗憾：

好多想达成的成就没有实现，也有好多话没说清楚。感觉要是认真写，可能就变成满满忏悔录了。
<没能温柔一点> 每每听到柔和舒缓的音乐，内心都是深情又安静的自己。很喜欢自己温柔的时候，但是一旦跟外界打交道，温柔立马就成为幻影。
<不够洒脱> 按照中国人平均寿命来算的话，现在也是正值壮年。可是却总有一种，过了30岁，一切都将不复存在的错觉。这一年常常听着歌就泪流满面，可能是想得太复杂了。不喜欢羁绊，又想要自己良善。

2019 的计划和决心

一、2019 计划
  • 多看书
    < 18年的没看完的书努力看完，19年多买书>
    <书看完要记得整理笔记，写写感想>
  • 养成适合自己的运动习惯
    <18年就想要的美腿，加油啦>
    <街舞很酷，跆拳道也不错>
  • 培养兴趣爱好
    <尤克里里学习>
  • 好好说话
  • 多出去走走
    <去 3个以上的地方>

写太多也没用，讲真，第一条能做好久真的很棒棒了，估计能缓解我一大部分的焦虑。fighting!!!

二、做困难的事

前端的框架帮忙做了太多的封装，简简单单就变成了码农。学会调试框架，去解决别人不能解决的问题。2019 做困难的事。
虽然罗列那么一堆计划，但是还是希望自己开开心心，乐呵呵的。
